\begin{dfn}[Ordered Ring]
We say that a ring \(R\) is \emph{ordered}\index{ordered ring} if there is a total order relation \(\leq\) on \(R\) which satisfies the following properties.
\begin{proplist}
\item If \(a,b,c \in R\) such that \(a \leq b\), then \(a+c \leq b+c\).
\item If \(0 \leq a\) and \(0 \leq b\), then \(0 \leq ab\).
\item \(0 \leq 1\).
\end{proplist}
\end{dfn}

\begin{prop}
Every commutative ordered ring is an integral domain with characteristic zero.
\end{prop}
